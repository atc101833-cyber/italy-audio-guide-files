更新方式：
1. GitHub Desktop 按 Show in Finder。
2. 把以下三個檔案放進 repository 根目錄：
   index.html
   service-worker.js
   offline-use-design.webp
3. index.html 與 service-worker.js 選擇取代。
4. 原本的圖片與 MP3 不要刪除。
5. Summary 輸入：
   Redesign offline download interface
6. Commit to main → Push origin。
7. 等 GitHub Pages 更新後，完全關閉 Safari 或桌面導覽 App，再連網重新開啟。
